🎯 HOW IT WORKS (The Magic!)
═══════════════════════════════════════════════════════════════════════════

CAMERA DETECTS:
  Camera → YOLOv5 Model → Detection Results → Statistics

STATISTICS SYNC TO UI:
  Every 1 second:
  • Total persons detected
  • Count with helmet
  • Count with vest
  • Count with glasses
  • Compliance percentage (0-100%)

SIMULATION RESPONDS:
  Every 2 seconds:
  • Gets detection compliance level
  • Activates GREEN LED if >80%
  • Activates RED LED if <50%
  • Updates motion detector indicator
  • Updates worker presence indicator

ALERTS GENERATED:
  Real-time:
  • Compliance < 50% = CRITICAL alert
  • Compliance 50-80% = WARNING alert
  • Red LED activation = Alert triggered
  • All logged with timestamps

EVERYTHING SYNCED IN ONE PAGE:
  Left Panel: Camera stream + controls
  Center: Detection stats + compliance meter
  Right: IoT simulation + LED indicators
  Bottom: Performance, Logs, Alert History
